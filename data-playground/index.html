Help & Shortcuts

Data Playground lets you paste JSON, CSV, or text, then run JavaScript to filter, transform, or explore it.

  • Paste Data: Use the first card to paste JSON, CSV, or text.
  • Write JS: Use data for parsed input and rawText for the exact original pasted text.
  • Run: Click or press Ctrl+Enter to run your code.
  • Switch Views: Use or to toggle output format.
  • Copy Output: copies the output.
  • Shortcuts: Ctrl+Enter (run), Esc (close modal), Ctrl+L (clear data).
